# Hermes Agent + Obsidian 打造第二大脑：14篇文章讲透第二大脑搭建

作者：sgr011215（安逸Ai），CSDN 新星创作者，发布于 2026-04-26。

## 核心思路

Obsidian = 本地存储 + 双向链接 + 知识图谱
Hermes Agent = 智能收集 + 自动整理 + 分层记忆
两者结合 = 真正的第二大脑

## 痛点
- 论文读过就忘
- 工具选型纠结
- 笔记散落各处（微信收藏、Notion、印象笔记）
- AI 助手没有记忆
