# 配补调退业务

服装零售行业商品库存管理的四大核心业务流程：**配货、补货、调拨、退货**。以 [[datamax|DataMax]] 平台在广州尚睿服装（加盟/直营/奥莱门店）的应用为参考。

## 业务背景

广州尚睿服装传统手工作业存在四大痛点：
- 补货不精准
- 调拨无优先级
- 配货效率低
- 退货无依据

通过系统化规则实现库存可视化与精细化管理。

## 四大模块

### 1. 补货（Replenishment）

**目标**：确保门店库存维持在安全库存水平以上，防止缺货。

**核心规则**：

| 要素 | 规则 |
|------|------|
| 安全库存 | 上市≤7天 = 上市库存；上市＞7天 = 每周一按近7天实际销量更新 |
| 目标库存 | 近7天平均销量 × X（可配置） |
| 补货量 | 目标库存 - 在仓库存 - 在途库存 - 配未发库存 |
| 仓库优先级 | 百世总仓 > 百世补货仓 > 门店仓 |
| 库存不足时 | 按店铺销量从高到低分配 |

**两种场景**：
- 场景 1：近 7 天销量 < 安全库存 → 补至安全库存
- 场景 2：近 7 天销量 > 安全库存 → 按销量补至目标上限（安全库存 + N）

### 2. 调拨（Transfer）

**目标**：将滞销店铺的库存调拨到缺货店铺，优化整体库存分布。

**核心规则**：

| 要素 | 规则 |
|------|------|
| 频率 | 大调拨（断码调拨）每周 1 次 |
| 需求计算 | 与补货逻辑一致（目标库存 = 7 天均销 × X） |
| 调出优先级 | 30 日无销售 > 14 日无销售 > 7 日无销售 |
| 店铺等级排序 | B > A > A+ > S > S+（C 店不参与） |
| 安全库存动态 | 7-14 天无销 → 1 件；21 天无销 → 0 |
| 调拨范围筛选 | 区域 > 城市 > 营销部门 > 辖区 > 店铺等级 |

**特殊规则**：
- 库存 > 1：可调出量 = 可用数 - 安全库存
- 库存 = 1：安全库存 = 0 的店铺按辖区→城市→营销部门→区域→店铺等级排序
- 近 7 日无销售店铺调出后保留 1 件；近 14 日无销售保留 0 件

### 3. 配货（Allocation）

**目标**：按订单或库存将商品分配到各门店。

#### 场景 1：期货配货（按订单配货）

| 要素 | 规则 |
|------|------|
| 筛选条件 | 订单编号、仓库、门店、商品颜色 |
| 门店优先级 | S+ > S > A+ > A > B > C |
| 仓库分配 | 加盟→加盟仓；直营→直营仓；奥莱→奥莱仓 |
| 配货逻辑 | 优先按订单需求满足，分批次按波段配货；同等级随机分配 |

#### 场景 2：增补款配货（按库存配货）

| 要素 | 规则 |
|------|------|
| 筛选条件 | 门店等级 |
| 核心规则 | 按 SKC 设置尺码比，按"配货手数"分配库存 |
| 尺码比设置 | 手动录入（单 SKC）或 Excel 导入（多 SKC） |
| 门店优先级 | S+ > S > A+ > A > B > C |

### 4. 退货（Return）

**目标**：将滞销商品退回仓库，释放门店库存空间。

| 要素 | 规则 |
|------|------|
| 频率 | 每周三执行 |
| 退货依据 | 实际销量 + 未来天气（如雨天减少户外服装库存） |
| 筛选维度 | 年份、波段、30/45/60 天不动销 |
| 退货量 | ≤ 可用库存数，默认等于可用库存，可手动调小 |
| 退货仓库 | 默认百世仓 |
| 加盟店 | 走渠道调拨单 |
| 直营店 | 走商店退货申请单 |

## 流程模式

四大模块遵循统一的操作流程：

```
筛选 → 自动计算 → 预览数据 → 确认数据 → 生成单据
```

均支持 Excel 导出/导入，便于离线调整后批量导入。

## 数据展示

支持两种展示模式：
- **横排**：店铺/仓库为行，SKU 为列，适合汇总查看
- **单列**：SKU + 店铺/仓库逐行展示，适合明细核对

## 门店分级体系

| 等级 | 补货 | 调拨（调出优先级） | 配货 | 退货 |
|------|------|-------------------|------|------|
| S+ | 高 | 最低（最后调出） | 最高 | 标准 |
| S | 高 | 低 | 高 | 标准 |
| A+ | 中 | 中 | 中 | 标准 |
| A | 中 | 中高 | 中 | 标准 |
| B | 低 | 最高（优先调出） | 低 | 标准 |
| C | 最低 | 不参与调拨 | 最低 | 标准 |

> 注意：调拨与配货的门店优先级方向相反。调拨优先从低等级门店调出（B 类优先），配货优先满足高等级门店（S+ 优先）。
