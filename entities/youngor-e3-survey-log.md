# 雅戈尔 E3+ 业务调研日志

雅戈尔集团 E3+ 业务中台迭代升级项目的业务调研日志（2023-10-30，V1.0），覆盖 5 大调研领域 + 6 类项目交付物。

## 关键信息

| 维度 | 说明 |
|------|------|
| **项目** | 雅戈尔业务中台迭代升级项目 |
| **日期** | 2023-10-30 |
| **版本** | V1.0 |
| **调研范围** | 主数据、库存、全渠道、电商业务（正向/逆向）、财务对账 |

## 调研内容（5 大领域）

### 1. 主数据及库存

| 调研项 | 现状 | 提升方向 |
|--------|------|---------|
| 业务组织架构 | E3 未维护复杂组织架构 | 通过用友中台获取 |
| 商品主数据 | 用友中台同步到 E3 | 中台推送至 E3+，增加补偿机制 |
| 其他主数据 | E3 未维护经销商/门店等 | 中台同步到 E3+ |
| 角色与权限 | — | 功能角色 + 数据角色 |
| 库存结构 | 全量/增量同步，中台分配后推 E3 | 升级 E3+ 后统一库存管理 |

### 2. 全渠道业务

| 场景 | 现状 | 涉及系统 |
|------|------|---------|
| 网订店发 | E3→中台→POS 门店发货 | POS、中台 |
| 网订店取 | 酷客多平台，下单带仓库信息 | POS、中台 |
| 店订仓发 | 暂无此场景 | — |
| 邻店发货 | A 店下单 B 店发，不经过 E3 | POS |

### 3. 电商业务-正向

| 调研项 | 现状 |
|--------|------|
| 涉及平台 | 淘宝、京东、唯品会、天虹、喵街、拼多多、抖音、快手、有赞、酷客多、杉杉奥莱、爱库存、好衣库、微盟、微信视频号 |
| 对接方式 | 酷客多/喵街/天虹/杉杉奥莱 → 用友中台（未来转 Adaptor） |
| 促销策略 | 满赠、买赠、排名送 |
| 仓库适配 | 同仓优先，电商仓优先 |
| 快递适配 | 按仓库+重量匹配 |

### 4. 电商业务-逆向

| 调研项 | 现状 | 提升方向 |
|--------|------|---------|
| 订单取消 | E3→中台→POS 逐级取消 | E3+ 直接发起取消，状态校验 |
| 退货退款 | 仓库 POS 扫描登记，中台回传 | E3+ 包裹匹配 + 自动生成退货单 |
| 换货 | 客服打标，入库后复制新单 | 自动下载换货申请，预占库存 |
| 仅退款 | E3 未下载，财务手工导入 | 下载仅退款申请，生成退款任务单 |
| 唯品会 | 全手工退单操作 | 自动下载退供单，系统入库 |

### 5. 财务对账

| 调研项 | 现状 |
|--------|------|
| 对账范围 | 除唯品会外所有渠道在 E3 对账 |
| 对账流程 | 财务下载账单 → 修改格式 → E3 导入对账 |
| 核心报表 | 订单出库报表、订单退货报表、应收账款明细表 |
