# 雅戈尔 E3+ 售前方案

雅戈尔集团 E3+ 升级方案的售前汇报材料（68 页 PPT），覆盖现状分析、提升优化、最佳实践三大板块。

## 关键信息

| 维度 | 说明 |
|------|------|
| **客户** | 雅戈尔集团 |
| **项目** | E3+ 升级方案 |
| **文档类型** | 售前方案汇报 PPT |
| **核心系统** | E3+（百胜 Adaptor + OMS + 库存中心 + 订单中心 + 财务中心） |

## 方案结构

### 一、现状分析

- **AS-IS 架构**：E3 OMS + 用友中台 + 道讯 DRP/POS + WMS
- **TO-BE 架构**：百胜 Adaptor（全网通） + E3+ 业务中台（订单中心/库存中心/商品中心/财务中心）
- **核心机会点**：自研商城业务快速扩展、全渠道业务高效协同

### 二、提升优化（四大方向）

#### A. 全域库存视图

| 能力 | 说明 |
|------|------|
| 四层库存逻辑 | 物理库存 → 逻辑库存 → 唯一码库存，分层处理不同业务 |
| 集团化多组织 | 法人公司 → 核算组织 → 库存组织 → 逻辑仓，明晰货权 |
| 共享 vs 独享 | 全渠道共享策略（按可用数同步，含警戒数/保留数） |
| 门店自提优化 | 流程从 4 步优化为 3 步，库存扣减统一由 E3+ 完成 |

#### B. 订单履约

| 能力 | 说明 |
|------|------|
| 正向链路 | 订单自动化审核、灵活适配策略（店订仓发/邻店发） |
| 逆向链路 | 统一退换货对接、自动退款处理 |
| 异常预警 | 运营设置类/买家行为类/库存不足类/门店推单异常，分类处理 |
| 业务场景 | 抖音指定区域发货、京东拆单、唯品会对接、GMV 分析 |

#### C. 财务对账

| 能力 | 说明 |
|------|------|
| 平台对账 | E3+ 标准对账单、自动对账、资金对账、销售对账 |
| 收入确认 | 发货确认收入规则 |
| 差异处理 | 对账差异处理、资金在途结转、发货在途结转 |
| 财务报表 | 资金三栏账、商店资金对账、自营商店结算单 |

#### D. 技术突破

| 能力 | 说明 |
|------|------|
| 架构升级 | E3（流程自动化）→ E3+（核心能力服务化） |
| 云原生 | 分布式微服务、共享化服务、场景化应用、多云支持 |
| 性能 | 千万级数据压测：729 单/秒，约 262 万单/小时 |

### 三、最佳实践

**小熊电器案例**：
- 月节流 26 万+，订单处理时效提升 4 天，月结节省 25 天
- 3 万+ 自动化处理，分销商自主下单/对账
- 分仓发货节省调拨费用约 100 万/年
- 实现供销链业财一体化

## 交付场景价值

| 场景 | 用途 |
|------|------|
