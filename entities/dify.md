# Dify

开源 LLM 应用开发平台，支持可视化编排 AI 工作流。

## 关键信息

- **类型**：LLMOps 平台
- **核心能力**：可视化工作流编排、RAG 管道、Agent 编排、模型管理
- **许可证**：Apache 2.0

## 与 Hermes Agent 集成

[[Hermes Agent]] 可通过 API 与 Dify 集成，将第二大脑接入专业工作流平台，实现复杂自动化编排。
